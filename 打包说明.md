# 打包成可执行文件（用于小红书分享）

## 方法一：使用 PyInstaller（推荐）

### 1. 安装 PyInstaller

```bash
pip install pyinstaller
```

### 2. 打包GUI版本

```bash
pyinstaller --onefile --windowed --name "Excel转VCF工具" --icon=app.ico excel_to_vcf_gui.py
```

参数说明：
- `--onefile`: 打包成单个可执行文件
- `--windowed`: 隐藏命令行窗口（GUI程序）
- `--name`: 设置可执行文件名
- `--icon`: 设置图标（可选）

### 3. 打包命令行版本

```bash
pyinstaller --onefile --name "excel_to_vcf" excel_to_vcf.py
```

### 4. 生成的文件位置

打包完成后，可执行文件在 `dist` 目录下。

## 方法二：使用 cx_Freeze

### 1. 安装 cx_Freeze

```bash
pip install cx_Freeze
```

### 2. 创建 setup.py

```python
from cx_Freeze import setup, Executable

setup(
    name="Excel转VCF工具",
    version="1.0",
    description="将Excel文件转换为VCF格式，导入iOS通讯录",
    executables=[Executable("excel_to_vcf_gui.py", base="Win32GUI" if sys.platform == "win32" else None)]
)
```

### 3. 打包

```bash
python setup.py build
```

## 方法三：使用 py2app（仅macOS）

### 1. 安装 py2app

```bash
pip install py2app
```

### 2. 创建 setup.py

```python
from setuptools import setup

APP = ['excel_to_vcf_gui.py']
DATA_FILES = []
OPTIONS = {
    'argv_emulation': True,
    'packages': ['pandas', 'openpyxl'],
}

setup(
    app=APP,
    data_files=DATA_FILES,
    options={'py2app': OPTIONS},
    setup_requires=['py2app'],
)
```

### 3. 打包

```bash
python setup.py py2app
```

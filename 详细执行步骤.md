### ✅ 任务4：用Canva制作配图（15分钟）

#### 步骤4.1：进入Canva
```
1. 访问：https://www.canva.cn
2. 登录你的账号
3. 点击"创建设计"
```

#### 步骤4.2：选择模板
```
1. 在搜索框输入"小红书"
2. 选择尺寸：1080 x 1350 px（重要！）
3. 浏览模板，选择一个简洁、干净的模板
4. 推荐风格：
   - 白色或浅色背景
   - 字体清晰易读
   - 有留白空间
5. ✅ 点击你喜欢的模板，开始编辑
```

#### 步骤4.3：制作第1张图（封面图）
```
1. 修改标题：
   - 删除模板原有文字
   - 输入AI生成的标题
   - 调整字体大小（要够大，能清晰看到）
   - 建议字号：40-60

2. 添加副标题（可选）：
   - 比如："3分钟搞定 | 完全免费"
   - 字号：24-32

3. 上传工具截图：
   - 点击"上传"
   - 选择 01-首页.png
   - 拖到画布上
   - 调整大小和位置

4. 调整布局：
   - 标题在上方
   - 截图在中间
   - 底部留白

5. ✅ 完成第1张图
```

#### 步骤4.4：制作第2张图（步骤图）
```
1. 点击"添加新页"
2. 选择空白页或简单模板
3. 添加标题："操作步骤"
4. 添加4个步骤（用文字）：
   第1步：打开网页
   第2步：上传Excel文件
   第3步：确认转换
   第4步：下载导入iPhone
5. 可以添加小图标装饰（Canva自带）
6. ✅ 完成第2张图
```

#### 步骤4.5：制作第3张图（效果展示）
```
1. 点击"添加新页"
2. 添加标题："效果展示"
3. 上传并添加：
   - 03-文件上传.png
   - 05-转换成功.png
4. 左右或上下排列
5. 添加文字说明（可选）
6. ✅ 完成第3张图
```

#### 步骤4.6：制作第4张图（总结引导）
```
1. 点击"添加新页"
2. 添加大标题："关注我"
3. 添加文字：
   "更多实用工具分享"
   "评论'1'获取工具链接"
4. 可以添加二维码或其他装饰
5. ✅ 完成第4张图
```

#### 步骤4.7：导出图片
```
1. 点击右上角"下载"
2. 选择"PNG"格式
3. 选择"多页"（所有页面）
4. 点击"下载"
5. 保存到：桌面/推广素材/第一次发布/
6. ✅ 检查：确保有4张图片
```

## ⚠️ 常见问题和解决方案

### 问题1：AI生成的文案不满意怎么办？
```
解决方案：
1. 在文心一言中说"再生成一个，标题更吸引人"
2. 或者说"第2段写得太专业，改得口语化一些"
3. 多生成几次，挑选最满意的
✅ 不要将就，一定要满意再用
```

### 问题2：Canva制作太慢怎么办？
```
解决方案：
1. 第一次会慢，熟练后很快
2. 保存模板，下次直接复用
3. 只修改标题和图片，其他不改
✅ 熟练后5分钟就能做好4张图
```

### 问题3：发布后没人看怎么办？
```
解决方案：
1. 检查标签是否添加
2. 检查话题是否选择
3. 检查发布时间（推荐晚8-10点）
4. 坚持发布，至少7天后看效果
✅ 不要着急，慢慢积累
```

### 问题4：不会剪辑视频怎么办？
```
解决方案：
1. 先从简单的录屏开始
2. 剪映有自动字幕功能，很简单
3. 跟着本文档的步骤一步步来
4. 或者先专注图文，视频可以后面再做
✅ 不要被视频吓到，其实很简单
```

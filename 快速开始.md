# 🚀 快速开始指南

## 3分钟快速上手

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

### 第二步：启动工具

**方式A：图形界面（最简单）**
```bash
./启动工具.sh
# 或
python3 excel_to_vcf_gui.py
```

**方式B：命令行**
```bash
python3 excel_to_vcf.py 你的文件.xlsx
```

### 第三步：导入iPhone

1. 将生成的VCF文件发送到iPhone（邮件/AirDrop/iCloud）
2. 在iPhone上打开VCF文件
3. 点击"添加到通讯录"

✅ 完成！

## 💡 常见问题

**Q: 需要安装Python吗？**
A: 是的，需要Python 3.7+。如果不想安装Python，可以打包成可执行文件。

**Q: Windows可以用吗？**
A: 可以！GUI版本和命令行版本都支持Windows。

**Q: 支持哪些Excel格式？**
A: 支持 .xlsx 和 .xls 格式。

**Q: 生成的VCF文件多大？**
A: 一般每个联系人约200-500字节，1000个联系人约200KB-500KB。

**Q: 可以批量导入多少联系人？**
A: 理论上没有限制，已测试过1000+联系人，速度很快。

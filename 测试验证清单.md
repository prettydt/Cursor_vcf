## 🔧 问题排查

### 网站无法访问
1. 检查GitHub Pages设置
2. 确认分支名称正确
3. 等待几分钟让部署生效
4. 清除浏览器缓存

### 功能不工作
1. 打开浏览器开发者工具
2. 查看Console错误信息
3. 检查Network标签的资源加载
4. 确认XLSX库正确加载

### PWA无法安装
1. 确保使用HTTPS
2. 检查manifest.json格式
3. 验证所有图标路径正确
4. 查看浏览器控制台错误

### SEO不生效
1. 等待几天让搜索引擎抓取
2. 确认robots.txt可访问
3. 提交sitemap到Google
4. 检查页面是否被正确索引

# 🚀 一键部署指南

本指南提供最简单、最快速的部署方法，让你的工具在5分钟内上线。

---

## ⚡ 方法一：GitHub Pages（最简单，推荐新手）

### 优势
- ✅ 完全免费
- ✅ 无需额外配置
- ✅ 自动HTTPS
- ✅ 无流量限制

### 步骤（3分钟）

1. **确保代码在GitHub仓库中**
   - 你的代码已经在 `https://github.com/prettydt/Cursor_vcf`

2. **开启GitHub Pages**
   ```
   仓库页面 → Settings → Pages → Source → 选择 main 分支 → Save
   ```

3. **访问网站**
   ```
   https://prettydt.github.io/Cursor_vcf/
   或
   https://prettydt.github.io/Cursor_vcf/index.html
   ```

4. **（可选）绑定自定义域名**
   - 在 Settings → Pages → Custom domain 中输入你的域名
   - 在域名DNS设置中添加CNAME记录指向 `prettydt.github.io`

### 更新网站
```bash
# 只需正常提交代码，网站会自动更新
git add .
git commit -m "更新内容"
git push
# 等待1-2分钟，网站自动更新
```

## 🚀 方法二：Vercel（最专业，推荐）

### 优势
- ✅ 全球CDN加速
- ✅ 自动部署（Git push即发布）
- ✅ 免费SSL证书
- ✅ 100GB免费流量/月
- ✅ 访问速度快

### 步骤（5分钟）

#### 网页操作（最简单）
1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 选择 `Cursor_vcf` 仓库
5. 点击 "Deploy"
6. 完成！获得网址（如：`https://cursor-vcf.vercel.app`）

#### 命令行操作
```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 在项目目录运行
cd /path/to/Cursor_vcf
vercel

# 3. 按提示登录并部署
# 4. 获得网址
```

### 绑定自定义域名
```bash
# 在Vercel项目设置中
Settings → Domains → 添加域名

# 然后在你的域名DNS中添加：
CNAME: www → cname.vercel-dns.com
A: @ → 76.76.21.21
```

## 🌟 方法三：Netlify

### 优势
- ✅ 拖拽部署
- ✅ 100GB免费流量
- ✅ 表单处理
- ✅ 即时预览

### 步骤（5分钟）

1. 访问 https://netlify.com
2. 注册/登录
3. 点击 "Add new site" → "Import an existing project"
4. 连接GitHub，选择 `Cursor_vcf` 仓库
5. 点击 "Deploy site"
6. 完成！获得网址（如：`https://cursor-vcf.netlify.app`）

### 或者拖拽部署
1. 访问 https://app.netlify.com/drop
2. 直接拖拽整个项目文件夹
3. 立即部署完成

## 🎯 推荐方案

### 新手 → GitHub Pages
- 最简单，3分钟上线
- 完全免费
- 适合个人项目

### 追求速度 → Vercel 或 Cloudflare Pages
- 全球CDN加速
- 访问速度快
- 适合有一定技术基础的用户

### 需要额外功能 → Netlify
- 支持表单提交
- 即时预览
- 适合需要后端功能的项目

## 🔧 部署后的配置

### 1. 更新manifest.json中的start_url
```json
{
  "start_url": "https://你的实际网址/index.html"
}
```

### 2. 更新index.html中的og:url
```html
<meta property="og:url" content="https://你的实际网址">
```

### 3. 添加Google Analytics（可选）
```html
<!-- 在<head>中添加 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

## ⚠️ 常见问题

### Q: 部署后页面显示空白？
A: 检查浏览器控制台是否有JavaScript错误，确保XLSX库能正常加载。

### Q: 如何更新网站内容？
A: 直接在GitHub中修改代码并提交，所有平台都会自动更新（通常1-2分钟）。

### Q: 可以绑定自己的域名吗？
A: 可以！所有平台都支持自定义域名，且提供免费SSL证书。

### Q: 流量超限怎么办？
A: GitHub Pages和Cloudflare Pages无限流量。Vercel和Netlify免费100GB/月，对个人项目足够。

### Q: 如何查看访问统计？
A: 
- Vercel/Netlify：平台自带分析面板
- GitHub Pages：需要添加Google Analytics
- 推荐使用Umami（开源、隐私友好）
